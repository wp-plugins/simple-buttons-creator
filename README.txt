== Description ==

A simple way to create buttons with many customizable options available. After a button is created, you can use a Simple Buttons Creator widget to display the button onto the page. This plugin has been tested with [Page Builder by SiteOrigin](http://wordpress.org/plugins/siteorigin-panels/ "Page Builder WordPress plugin").

== Installation ==

You can either:

1. Upload unzipped `simple-buttons-creator` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin `Simple Buttons Creator` through the 'Plugins' menu in WordPress.
1. The `Simple Buttons Creator` menu can be found in the main menu of WordPress admin.

Or:

1. Search `Simple Buttons Creator` in the Add New plugin section in WordPress admin and install it.
1. The `Simple Buttons Creator` menu can be found in the main menu of WordPress admin.
